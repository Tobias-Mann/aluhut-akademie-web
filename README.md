# aluhut-akademie.de

Statisch generierte Begleit-Website zum satirischen Kartenspiel **Aluhut-Akademie**.
Jede gedruckte Karte trägt einen QR-Code, der auf ihre Infoseite mit dem realen,
belegten Hintergrund der parodierten Erzählung führt.

- Dieses Repository enthält nur das **Build-Ergebnis** (GitHub-Pages-Hosting).
  Es wird aus einem privaten Quell-Repository generiert (`python -m render.website`);
  direkte Änderungen hier werden beim nächsten Deploy überschrieben.
  Issues und Pull Requests werden daher nicht angenommen.
- Satire-Hinweis: Die Seiten dokumentieren und parodieren Verschwörungserzählungen —
  sie bestätigen sie nicht.
